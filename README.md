# 📜 منصة دراسة تاريخ الأردن — تفاعلية

> أداة دراسة بصرية تفاعلية لكتب التاريخ الأردنية الأربعة. ملف HTML واحد، يعمل بدون إنترنت، RTL كامل، يدعم الوضع الداكن.

## 🎯 الفكرة

دراسة التاريخ من الكتاب الورقي صعبة — النص طويل، الأحداث متشعبة، والمعلومات مبعثرة. هذا التطبيق يحوّل **محتوى الكتب الأربعة الكامل** إلى تجربة دراسة بصرية: خط زمني تفاعلي، بطاقات منظمة لكل شخصية ومعاهدة ومفهوم، بحث شامل، وروابط ذكية تربط الأحداث عبر الكتب الأربعة.

## 📊 المحتوى

| المقياس | العدد |
|---------|------|
| الكتب | **4** |
| الوحدات | **9** |
| الدروس | **42** |
| أحداث الخط الزمني | **351** |
| البطاقات الدراسية | **380** |
| الفترة الزمنية | من **2.5 مليون سنة** إلى **اليوم** |

تغطي الكتب: العصور الحجرية، الأنباط، الفترة اليونانية والرومانية والبيزنطية، الفتح الإسلامي، الأموي، العباسي، الفاطمي والأيوبي، المملوكي، العثماني، الثورة العربية الكبرى، تأسيس الإمارة، استقلال المملكة، الحياة السياسية والاقتصادية والاجتماعية الحديثة، والقضية الفلسطينية.

## ✨ الميزات

### الأساسية
- 🌳 **شجرة تنقل** كاملة: 4 كتب ← 9 وحدات ← 42 درس قابلة للطي
- ⏳ **خط زمني** أفقي لكل درس بألوان متدرجة حسب الحقبة
- 🗂️ **بطاقات دراسية** بأيقونات وألوان حسب النوع (شخصية، معاهدة، مفهوم، حركة...)
- 🔍 **بحث فوري** في الدروس بالشريط الجانبي
- 🌗 **الوضع الداكن** يُحفظ تلقائياً
- 📱 **متجاوب** يعمل على الموبايل بكفاءة كاملة (الـ Sidebar تصير Drawer)
- 🅰️ **RTL كامل** بخط Tajawal العربي

### المتقدمة
- 🔎 **بحث شامل** عبر الكتب الأربعة بضغطة `Ctrl+K` — يبحث في أسماء الأشخاص، التواريخ، الأحداث، المعاهدات، المفاهيم. نتائج مظلَّلة ومجمَّعة.
- 🔗 **روابط ذكية** بين الكتب: عند فتح بطاقة "الشريف الحسين بن علي" تظهر كل ظهوراته في الكتب الأخرى بضغطة واحدة.
- 🌐 **الخط الزمني الشامل**: عرض الـ 351 حدث على خط زمني عمودي واحد مقسوم لـ 11 حقبة تاريخية، مع فلاتر حسب النوع والكتاب.
- ⚖️ **مقارنة جنباً لجنب**: أضف بطاقتين (مثلاً الأمويين والعباسيين) وقارن تفاصيلهما في عمودين متجاورين.
- 🖨️ **ورقة مراجعة قابلة للطباعة**: زر طباعة يولّد ورقة A4 نظيفة بدون واجهات للدرس الحالي.

## 🚀 الاستخدام

### للطالب
1. افتح الرابط في المتصفح
2. اختر درساً من الشريط الجانبي
3. اقرأ الخط الزمني، استكشف البطاقات
4. استخدم `Ctrl+K` للبحث السريع عن أي معلومة
5. اضغط 🖨️ لطباعة ورقة مراجعة قبل الامتحان

### اختصارات لوحة المفاتيح
| الاختصار | الوظيفة |
|---------|---------|
| `Ctrl+K` أو `/` | فتح البحث الشامل |
| `Esc` | إغلاق أي نافذة منبثقة |
| `Ctrl+P` | طباعة ورقة المراجعة |

## 🛠️ التشغيل المحلي

ما تحتاج أي تنصيب. فقط:
```
افتح index.html بأي متصفح حديث (Chrome, Firefox, Edge, Safari)
```

كل الملف ذاتي الاحتواء (HTML + CSS + JS + البيانات) — حتى الخط العربي يُحمَّل تلقائياً من Google Fonts عند توفر الإنترنت، ولو ما فيه إنترنت بيشتغل بخط احتياطي.
